<h1 align="center">Lucide Studio</h1>
<h3 align="center">Lucide Studio is a development tool built with Next.js, made to help with creating icons for Lucide.</h3>
<p align="center">Currently live at <a href="https://studio.lucide.dev/">studio.lucide.dev<a/></p>

<br/>

## Contributing

### Install Modules

Make sure to run the following to install the modules needed

```bash
npm i --legacy-peer-deps
```

### Start a Dev Server

Run the following command:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
